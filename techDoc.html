HTML

HyperText Markup language, or HTML for short, is the standard markup language for webpages.

Below is the basic layout or "boilerplate" to get you started.

    <!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        </head>
    <body>
                        
    </body>
</html>

The first written line of code is not a tag but it is a declaration to the browser.

<!DOCTYPE html>

This specifies that the document type is written in html.

Tags

The first tag is the html tag. This is the root tag and is the container for all other tags used. In this we specify what language our file is wriiten in. In our case we set it to "en" which stands for english. This tag includes it's opening tag and a closing tag. 

 <html lang="en">
    ...
</html>

Next we have the head tag which holds meta tags and a title tag.

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

The meta element is part of the head section and contains the metadata, or the page description. This is used to describe the document. The first meta tag is for defining how the page should be encoded, in this case it is UTF-8. The second meta tag defines the viewport and its properties cause the dimensions to match width of the devices like on mobile. This is from the width=device-width. The initial-scale=1.0 is used to control the initial zoom level when the page is first loaded.

Our tag element within the head is the title tag. This tells the browser the title of the document you are creating. This is required for all HTML documents. Then we end off with our closing head tag.

The next tag we have is the body tag. This is where the main content of the webpage or document will be written. It contains, but is not limited to, a navigation bar, sidebar, and main content.

Finally we closing off with the closing html tag which indicates the end of the html document.

Attributes

Attributes are certain indicators that we add to tags to modify them. For example in the opening HTML tag:

<html lang="en">

In this case the attribute is lang="en". This says that the html document is written in english. Usually the attributes are written inside the opening tag with two parts separated by an equal sign (=).

<tag attribute="value"> ... </tag>
Content

Next we have the content of the html document. This is usually written inside the body tag.

In this we can usually find a

  • nav tag (short for navigation like the one on the side of this webpage)
  • main tag
  • sidebar tag
  • aside tag
  • footer tag
<body>
    <nav>
        ...
    </nav>

    <main>
        ...
    </main>

    <footer>
        ...
    </footer>
</body>

These are some of the basic tags that are used for the content of a webpage.

CSS

Cascading Style Sheets or CSS for short is a stylesheet used to style a markup language such as HTML. You would add styles targeting certain tags or attributes and then link it inside the head element. For example if your css file was named "stylesheet" it would be:

<link rel="stylesheet" href="stylesheet.css">

It can be used to change properties of things such as text color and weight. There are many more things that css can be used for. It is important to know some css so that you are able to style your HTML document for your webpage.
